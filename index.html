apache在wamp中创建虚拟主机

1.我现在用的是WampServer2.0i版本

在“httpd.conf”文件中查找：Include conf/extra/httpd-vhosts.conf，去掉前面的注释#。 如果安装在D盘那就是D:\wamp\bin\apache\apache2.2.8\conf\extra\httpd-vhosts.conf 用记事本打开

2.打开在C:\WINDOWS\system32\drivers\etc中的hosts文件中加上你要的域名,例如：

127.0.0.1    www.haha.com

3.修改httpd.conf文件：在文件内容的最下面加上如下 (下面的内容是两个虚拟主机的配置)

NameVirtualHost *:80 ServerAdmin riathinker@gmail.com
DocumentRoot d:/wamp/www/demo/ ServerName www.haha.com Options FollowSymLinks AllowOverride None Order allow,deny Allow from all

说明: ServerAdmin riathinker@gmail.com //管理员的邮箱 DocumentRoot d:/wamp/www/demo/ //网站目录 ServerName demo.com//域名 //目录的权限 Options FollowSymLinks AllowOverride None Order allow,deny Allow from all

也可以用里面自带的demo

ServerAdmin riathinker@gmail.com DocumentRoot "c:/wamp/www/demo" ServerName demo.com ServerAlias http://www.demo.com/ ErrorLog "logs/dummy-host.localhost-error.log" CustomLog "logs/dummy-host.localhost-access.log" common

另外还有一种比较简单的方法，就是想httpd.conf里最下端直接添加

NameVirtualHost 127.0.0.1

ServerAdmin riathinker@gmail.com DocumentRoot c:/wamp/www/demo ServerName localhost

ServerAdmin riathinker@gmail.com DocumentRoot c:/wamp/www/demo ServerName demo.com AllowOverride all Order Allow,Deny Allow from 127.0.0.1
